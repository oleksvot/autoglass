
* Install Python 3.11 https://www.python.org/downloads/. During installation, pick options "Add to PATH" and "Disable PATH length limit."
* Run install.bat from the script's directory
* Check if Google Chrome is installed and up to date.
* To start scraping, run start.bat. Note that this will remove old result files from the script's directory.
* If the process has been interrupted, close all windows related to the scraper, then run resume.bat.
* When done, your scraped data will be in the result.xlsx file.
